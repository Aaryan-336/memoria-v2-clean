# Design System

## Philosophy

Memoria should feel calm, academic, and intelligent.

The product should reduce cognitive load.

Avoid flashy interfaces.

## Design Principles

- Minimal.
- Clean.
- Modern.
- Student-first.
- Accessible.

## Color Palette

Primary:
- White backgrounds.

Accent:
- Deep blue.

Highlights:
- Soft indigo.

Success:
- Green.

Errors:
- Red.

## Typography

Headings:
- Modern sans-serif.

Body:
- Highly readable.

Large spacing between sections.

## Components

Cards:
- Rounded corners.
- Soft shadows.

Buttons:
- Simple.
- High contrast.

Forms:
- Minimal validation friction.

Search:
- Prominent and accessible.

## Dashboard

Homepage:
- Recent lectures.
- Study streaks.
- Continue learning section.

Lecture View:
- Transcript.
- Notes.
- Flashcards.
- Quiz tabs.

Chat Interface:
- Source citations.
- Conversation history.

## Responsive Design

Must support:
- Mobile
- Tablet
- Desktop

Mobile-first approach.

## Animations

Use sparingly.

Animations should communicate state changes, never distract from learning.
