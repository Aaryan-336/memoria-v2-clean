# Product Requirements Document

## Target Users

Primary:
- College students
- Competitive exam aspirants

Secondary:
- Teachers
- Researchers

## Functional Requirements

FR-1 Upload lecture audio.

FR-2 Upload PDFs.

FR-3 Upload PPT slides.

FR-4 Upload images of handwritten notes.

FR-5 Transcribe audio.

FR-6 Generate structured notes.

FR-7 Generate flashcards.

FR-8 Generate quizzes.

FR-9 Semantic search across uploaded content.

FR-10 Ask questions about lectures.

FR-11 Authentication.

FR-12 Subscription tiers.

FR-13 Upload YouTube links.

FR-14 Extract transcripts from YouTube videos.

FR-15 Generate notes from YouTube content.

FR-16 Create flashcards and quizzes from YouTube videos.

FR-17 Allow users to chat with YouTube lectures using RAG.

## Non-Functional Requirements

- Average response time under 3 seconds.
- Secure handling of private educational data.
- Mobile responsive UI.
- Scalable architecture.
- Fault-tolerant processing.
